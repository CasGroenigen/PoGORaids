# PoGORaids

**PoGORaids.net** — A community hub for Pokémon GO trainers.

## Features
- 🏆 Leaderboards
- 🗺️ Community Map & Nest Map
- 🎁 Vivillon Gift Exchange
- 📅 Event Calendar
- 📊 Infographics
- 📰 Latest Pokémon GO News
- 💬 Discord community integration
- 🌐 Multilingual: English, Spanish, Portuguese, Dutch, German

## Tech Stack
Plain HTML, CSS, and JavaScript — no frameworks required. Designed for easy CMS/API integration.

## Getting Started
Open `index.html` in any browser, or serve with any static file server.
